# colex

The Programs you have to program at the end of Chapter 8 of the "The Rust Programming Language" Book

## Installation:
```sh
#Clone the git repo
git clone https://github.com/revellan/colex
cd colex

#build from source
make

#install to /usr/local/bin
sudo make install
```

## Uninstall:
```sh
sudo rm /usr/local/bin/colex
```
